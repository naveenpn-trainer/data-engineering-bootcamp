# Data Engineering Bootcamp

## Prerequisites

1. Download Git Bash
2. Download 7Zip

## Set Up Instructions

1. Open Git Bash
2. Execute the below commands

   ```
   $>  cd c:
   $> git clone https://github.com/naveenpn-trainer/data-engineering-bootcamp.git
   ```

3. Download the [environments](https://drive.google.com/file/d/1J0nHyjzdk8eBGzbX32EBpwZg4P2vm6NQ/view?usp=drive_link) package to c:\data-engineering-bootcamp; Right click environments.zip and Extract Here

4. Download the [softwares](https://drive.google.com/file/d/13Gx-to38lxxKjNIvOXa8vgxd_hUGUYh8/view?usp=drive_link) package to c:\data-engineering-bootcamp; Right click environments.zip and Extract Here

5. Extract the downloaded file **into the project root folder** so that the structure looks like:

   ```
   data-engineering-bootcamp/
   │
   ├── environments/
   ├── data-engineering-lab/
   ├── projects/
   ```
